📋 How to get the Google Sheets URL

  1. Open your Google Sheet
  2. Click File → Share → Publish to web
  3. First dropdown → select your sheet tab name (e.g. Sheet1)
  4. Second dropdown → select Comma-separated values (.csv)
  5. Click Publish → confirm → copy the URL
    https://docs.google.com/spreadsheets/d/e/…/pub?gid=0&single=true&output=csv
  6. Paste that URL in the box above and click Load
⚠ You must use Publish to web — NOT the regular "Share" button. They are completely different. The regular share link will NOT work.
